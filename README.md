
# PDF RAG Chatbot

An AI-powered PDF Question Answering chatbot built with Flowise AI and OpenAI using Retrieval-Augmented Generation (RAG).

## Features
- Upload and process PDF documents
- Context-aware question answering
- RAG architecture
- Flowise AI workflow
- OpenAI integration

## Tech Stack
- Flowise AI
- OpenAI
- RAG
- Vector Database
- PDF Processing

## Use Case
This chatbot allows users to upload PDF documents and ask natural language questions. It retrieves relevant information from the document and generates accurate responses.
